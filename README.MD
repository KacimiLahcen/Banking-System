# Système de Gestion Bancaire – PHP OOP (CLI)

## Description
Ce projet est une application bancaire en ligne de commande (CLI) développée en PHP.
Elle permet de gérer des clients, des comptes bancaires (courant et épargne) et des transactions
(dépôts et retraits) en utilisant la Programmation Orientée Objet (OOP) et PDO avec MySQL.

Le but principal du projet est d’apprendre :
- L’OOP en PHP
- L’héritage et le polymorphisme
- La persistance des données avec PDO
- Les transactions SQL (commit / rollback)

---

## Technologies utilisées
- PHP (CLI)
- MySQL
- PDO
- XAMPP (Windows)
